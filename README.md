# R-project-to-load-JSON-Data
A small Data analysis of the tripadvisor dataset using R.


This project aims at:

1.loading the JSON data into R

2.preparing the data to be used for analysis.

3.analysing the facts like:

  i.All the distinct cities present in the dataset
  
  ii.cities having the number of hotels greater than 30
  
  iii.creating a excel or csv file containing the data in the format(hotelname,hotelid,address,price)
